# MenuNavigator

MenuNavigator app in React Native and ES6 syntax

## How to run

1. Run `npm install`
2. Open ios/MenuNavigator.xcodeproj and hit run in Xcode.

## Platform

- iPad Pro
- iOS 9.2

## Other Dependencies

- [react-native-side-menu](https://www.npmjs.com/package/react-native-side-menu)
